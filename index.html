Zen Garden – Sztuka Stylu CSS

Eksperymentuj z wyglądem, nie zmieniając struktury HTML.

1. Idea projektu

Co to jest CSS Zen Garden?

To inicjatywa pokazująca, że wygląd strony można całkowicie zmienić za pomocą CSS, bez ingerencji w HTML.

Każdy styl może nadać stronie unikalny klimat i układ.

Dlaczego to ważne?

Separacja treści od prezentacji to podstawa nowoczesnego web designu.

2. Struktura dokumentu

Nagłówki i akapity

Używamy semantycznych znaczników takich jak <header>, <main> i <footer>.

Listy i sekcje

Listy pomagają uporządkować treści, a sekcje grupują powiązane tematy.

  • Semantyczna struktura HTML
  • Responsywny układ
  • Dwa style CSS

3. Responsywność

Dostosowanie do urządzeń

Strona dopasowuje się do szerokości ekranu, zapewniając czytelność na każdym urządzeniu.

Nowoczesne techniki

Użyto Flexbox i Grid, aby układ był elastyczny i atrakcyjny wizualnie.
